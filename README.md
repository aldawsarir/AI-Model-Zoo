# AI Model Zoo (Mini Version)

AI Model Zoo is a simple Python console-based application that simulates a repository for managing artificial intelligence models.

## Features
- Add new AI models
- Update model task, accuracy, or training status
- Remove existing models
- Display all models in a formatted table
- Exit system using a menu-driven loop
